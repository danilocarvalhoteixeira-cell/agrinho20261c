🌱 Agro Forte – Futuro Sustentável
Descrição

O Agro Forte é um site informativo desenvolvido com HTML, CSS e JavaScript que visa conscientizar sobre a importância da agricultura sustentável, mostrando como equilibrar produção agrícola e preservação ambiental.

O site apresenta informações sobre práticas responsáveis no campo e utiliza recursos interativos para envolver o usuário.

Funcionalidades
Design Responsivo: se adapta a diferentes tamanhos de tela.
Conteúdo Interativo: botão “Saiba Mais” que revela informações extras sobre sustentabilidade.
Foco em Sustentabilidade: destaca práticas como agricultura de precisão, preservação da biodiversidade e uso racional da água.
Tecnologias Utilizadas
HTML5 – Estrutura da página.
CSS3 – Estilização e efeitos visuais.
JavaScript – Interatividade (botão “Saiba Mais”).
